About This Website

This website was made by five students from DCU's Computer Applications and Enterprise Computing courses.

Below, you'll find details of the contribution each person made to the site, as well as some mistakes we made along the way.


Scott Brady

I was reponsible for the Iceland section of the site. I also played a major role in the overall design of the site, including the homepage and the navigation bars.

Some mistakes I made were:

  1. Using accents in the filenames of my sound files:

    This caused issues when I checked my language page on the W3C HTML validator. To fix this, I had to remove the accents from the filenames.

  2. Using an "ol" tag on the language page:

    This also caused issues with the validator as I had placed other elements such as paragraps and audio with in the "li" elements, which I learned was not a correct thing to do. To fix it, I replaced the list with several article tags instead.

Jack MacNamara

I was reponsible for France segement of this project.

Some mistakes I made were:

  1. Using uppercase when naming files.

    This caused a major issue when i was linking files due to serves being case sensitive. It took a while to figure out the mistake but eventually I fixed the issue.

  2. Forgetting to close off the tags in my HTML file.

    This cause a major problem when running my code in chrome. Most paragraphs we not in sequence and all over the place due to not closing off tags.

Okikiola Sanni

I was reponsible for the Nigeria section of the site.

Some mistakes I made were:

  1. Closing tag:

    I forgot to close the "table" tag, so this made my table appear down the page, below the footer. So I had to close the tag to fix the issue.

  2. Wrong Directory:

    I used a wrong directory for the image part of the Nigeria section. I used "image" instead of "img", so the images did not appear.
